# MILESTONES

## Milestone A

Project skeleton, planning memory files, architecture notes, and mock backend foundation.

## Milestone B

MCP v1 tool contracts, normalized document state, deterministic mock backend, and mock-focused tests.

## Milestone C

End-to-end validation of the v1 flow in mock mode, including stronger failure-path coverage.

## Milestone D

Scaffold and implement the real Windows SolidWorks adapter boundary.

## Milestone E

Run the first real validation pass on SolidWorks 2022 and document verified behavior.

## Milestone F

Extend capability scope toward richer modeling flows such as assemblies, drawings, and advanced features.
